### Beschreibung ###
Erweitert die Hersteller um eine Langbeschreibung

### Installation ###
Zuerst müssen die SQL-Statements aus der Datei install.sql in der Datenbank (phpMyAdmin oder anderer SQL-Client oder Oxid Admin unter Service->Toosl)
ausgeführt werden. Alle Dateien aus dem Ordner copy_this in den Shop kopieren. Im Admin unter Erweiterungen -> Module das neue Modul Hersteller-Beschreibung
aktivieren. Danach den tmp-Ordner mit einem FTP-Programm leeren und die Views updaten (Service -> Tools). 
Im Admin unter Hersteller gibt es nun einen neuen Reiter namens Langtext.
